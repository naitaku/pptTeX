pptTeX

ダウンロード

必要なソフトウェア (作者の動作確認環境)

  • Windows 8 64bit
  • PowerPoint 2013
  • TeXLive 2012
  • BaKoMa TrueTypeフォント（2004年修正版）
    -- http://argent.shinshu-u.ac.jp/~otobe/
  • pstoedit
    -- http://sourceforge.net/projects/pstoedit/
  • Ghostscript 9.07 32bit
    -- http://www.ghostscript.com/download/gsdnld.html

セットアップ

  1. 「必要なソフトウェア」をインストールしておく．
  2. pptTeX.exeを実行しインストール．
  3. PowerPointを起動すると設定画面が表示されるので，適切な値を入力．
  4. PowerPointにpptTeXのメニューが現れます．
